# Contributing to AI Memory System

This project is currently maintained by Keith Costello and Amy Costello for personal use only. While we've made the code available under the MIT license, we are not actively seeking external contributions at this time.

## Usage and Adaptation

You are welcome to:
- Fork this repository for your own use
- Adapt the system for your own projects
- Share your experiences using the system

## Personal Use License

Though licensed under MIT, this project is intended for personal use. If you wish to incorporate this into commercial products or services, please contact the repository maintainers.

## Contact

For questions about usage or implementation:
- Create an issue in the repository
- Reference the documentation in README.md and BASELINE_SETUP.md

## Code of Conduct

When interacting with this repository:
- Be respectful of the maintainers' time
- Focus discussions on technical aspects of the memory system
- Provide clear reproduction steps if reporting issues

Thank you for your interest in the AI Memory System.
